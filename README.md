# VR/AI Mental Health Companion for Seniors

A production-ready web application that provides gentle AI-assisted emotional support, mood tracking, and accessible digital wellness experiences designed specifically for older adults.

This project demonstrates how AI-powered interaction, accessible design, and modern web technologies can be combined to support mental well-being in an intuitive and inclusive way.
